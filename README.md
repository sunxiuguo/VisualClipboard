## 介绍

VisualClipBoard 是一款剪贴板工具，能够记录您复制、剪切的所有图片、文本、超文本历史。

## 支持格式

-   支持文本
-   支持图片
-   支持 excel 表格
-   支持网页内容

## 特性

-   存储 7 天的数据，过期数据自动清理
-   关键字筛选
-   列表瀑布流
-   列表缩略显示，点击弹窗显示详情
